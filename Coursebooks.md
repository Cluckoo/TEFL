## Using the coursebook well
The coursebook will have too much material to cover completely. You must select the most relevant parts to both keep students interested and not overload them!

| Problem                                          | Solution                                                                        |
| ------------------------------------------------ | ------------------------------------------------------------------------------- |
| Main topic of unit is boring                     | Choose a different topic with:<br>- another coursebook<br>- your own materials  |
| Students already know main grammar point of unit | Skip it                                                                         |
| Reading texts have lots of unfamiliar vocab      | Pre-teach vocabulary in lesson before or with homework<br>OR<br>Substitute text |
| Unit exercises don't recycle language            | Create your own revision activities                                             |
| Grammar point poorly explained                   | Create own materials to present grammar                                         |
| Listening dialogue is inauthentic                | Find subsitute<br>OR<br>Record your own dialogue with colleague                 |

If an activity is boring but still fits the aim, be creative and alter it to make it more interactive! Always try to make coursebook activities more interactive!

Ensure you make the coursebook feel personalised by supplementing the activities. For example, if a speaking activity has topic X, start a classroom discussion on topic X before beginning the activity. Also, you can change the materials to realia!
